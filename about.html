About Umbrella
Built for the mountains.
Because the mountains don't wait.
A multi-hazard early warning system for the Indian Himalayan Region connecting live satellite data to village-level disaster response.
The Problem

The Indian Himalayan Region faces accelerating disasters — glacial lake outbursts, flash floods, forest fires and earthquakes. Umbrella connects live hazard data to village-level awareness so responders can understand risk quickly.

What Umbrella Does
🌧
Flood & GLOF Risk
48hr rainfall monitoring and glacial lake flood travel analysis.
🌍
Seismic Monitor
Live earthquake detection using global seismic feeds.
🔥
Forest Fire Tracker
Satellite fire detections across Himalayan forests.
📊
Village Vulnerability
Risk scoring using NDMA hazard vulnerability exposure framework.
